\doxysubsection{Detailed Description}
Represents the client\textquotesingle{}s encrypted message request packet. 

This structure defines the layout of the client message request packet. It contains\+:
\begin{DoxyItemize}
\item A common HKDS packet header.
\item The client\textquotesingle{}s Key Serial Number (KSN).
\item The encrypted message.
\item An optional authentication tag (MAC) appended to the message. 
\end{DoxyItemize}
