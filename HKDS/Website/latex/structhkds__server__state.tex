\doxysubsection{Detailed Description}
Contains the HKDS server state. 

This structure represents the state of an HKDS server instance. It includes\+:
\begin{DoxyItemize}
\item {\ttfamily ksn\+:} The client\textquotesingle{}s Key Serial Number (KSN).
\item {\ttfamily mdk\+:} A pointer to the master key set used for deriving keys.
\item {\ttfamily count\+:} The token or transaction count.
\item {\ttfamily rate\+:} The output rate for the key derivation function (PRF). 
\end{DoxyItemize}
