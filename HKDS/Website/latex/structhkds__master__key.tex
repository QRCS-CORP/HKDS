\doxysubsection{Detailed Description}
Contains the HKDS master key set. 

The master key set is comprised of\+:
\begin{DoxyItemize}
\item {\ttfamily bdk\+:} The Base Derivation Key used to derive client embedded keys.
\item {\ttfamily stk\+:} The Secret Token Key used to generate device tokens.
\item {\ttfamily kid\+:} The Key Identity, which uniquely identifies the master key set. 
\end{DoxyItemize}
