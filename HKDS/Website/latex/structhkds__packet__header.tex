\doxysubsection{Detailed Description}
The primary header for all HKDS messages. 

This structure represents the common header present at the beginning of every HKDS packet. It contains\+:
\begin{DoxyItemize}
\item {\ttfamily flag\+:} The packet type.
\item {\ttfamily protocol\+:} The protocol identifier (indicating the SHAKE variant).
\item {\ttfamily sequence\+:} The packet sequence number.
\item {\ttfamily length\+:} The total length of the packet (including the header). 
\end{DoxyItemize}
