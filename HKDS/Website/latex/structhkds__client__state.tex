\doxysubsection{Detailed Description}
Contains the HKDS client state. 

This structure holds all state information required by the HKDS client. It includes\+:


\begin{DoxyItemize}
\item {\ttfamily edk\+:} The Embedded Device Key used for cryptographic operations (size defined by {\ttfamily HKDS\+\_\+\+EDK\+\_\+\+SIZE}).
\item {\ttfamily ksn\+:} The Key Serial Number containing the device identity and transaction counter (size defined by {\ttfamily HKDS\+\_\+\+KSN\+\_\+\+SIZE}).
\item {\ttfamily tkc\+:} The Transaction Key Cache, an array of keys (each of size {\ttfamily HKDS\+\_\+\+MESSAGE\+\_\+\+SIZE}) used for message encryption and authentication. The total number of keys is defined by {\ttfamily HKDS\+\_\+\+CACHE\+\_\+\+SIZE}.
\item {\ttfamily cache\+\_\+empty\+:} A boolean flag indicating whether the key cache has been exhausted. 
\end{DoxyItemize}
